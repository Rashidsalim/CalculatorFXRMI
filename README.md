### Command Variables
- Execute the Java Compiler
```
$ javac --module-path "C:\javafx-sdk-11.0.2\lib" --add-modules javafx.controls,javafx.fxml -classpath "D:\Repo\CalculatorFXRMI\*.java" .java
```
- Start RMI Registry
```
$ start rmiregistry
```
- Start the Server
```
$ java Server
```
